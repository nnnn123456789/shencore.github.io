最早只是一个SC的实验性项目，总计10个页面三四个附加内容，所以没太考虑这些问题，因此SC部分以及与SC同步的FC 3rd部分的文件结构同样没改，堆积下来就有点乱了= =不过已经不打算继续扩充了，眼下已经够用，也不打算重写架构了<br>
曾经有个写碧轨的计划，架构采取了科学得多的形式，完全足以满足此类RPG游戏单体攻略的需求，可惜我不想继续写了= =
